# SZE-MOSZE-2021-Signal
## Akasztófa játék
A program egy klasszikus akasztófa játékon alapul. Két játékos játszhat egyszerre. A kitalálandó szavak egy txt-ben tárolódnak. Karakterek megadásával tudnak tippelni.
A játék véget ér, ha kitalálandó szó minden karaktere megtippelésre került. Az utolsó karakter kitaláló személy nyer.
